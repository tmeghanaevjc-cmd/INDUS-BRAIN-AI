## 📌 Overview

INDUS BRAIN AI is an AI-powered Industrial Knowledge Intelligence Platform that centralizes engineering documents, maintenance records, SOPs, inspection reports, and safety manuals into a single intelligent knowledge base.

Using Retrieval-Augmented Generation (RAG), OCR, and Large Language Models, engineers can search documents, ask questions in natural language, receive AI-generated answers with document references, and improve operational efficiency.

- 📄 AI Document Search
- 🤖 AI Chatbot
